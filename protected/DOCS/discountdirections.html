Notes on Adding a new Discount Code
Discount Code: The Discount Code can be any alpha numeric string. Is NOT case sensitive.
Subtotal Range: The discount will be applied only if the subtotal falls in the range entered.
Examples (10.99-25.99, 10-, -100) The first applies the discount if it is between 10.99 and 25.99. The second applies it if the subtotal greater than 10 and the third applies it if the subtotal is less than 100.
Quantity Range: The discount will be applied only if the quantity ordered falls in the range entered.
Examples (10-25, 10-, -100)
Measured Value Range: The discount will be applied only if the weight or the items ordered falls in the range entered.
Examples (10-25, 10-, -100)
Discount: The Discount can be any amount in percentages or hard dollar amounts.
Examples (5%, 15.00, 20)
Discount Date: Enter the dates the discount is valid using the date form MM/DD/YYYY.
Examples (05/17/2003-05/17/2006, -01/01/2007, 12/31/2003-)
Comments: Enter comments to remind you of the discount or promtion. Use only alpha numeric characters.
